# Fuck Every Word

[VK] & [Telegram] bot generating 3 years worth amount of content.

Inspired by twitter bot [Fuck Every Word 2.0]

Images are searched via [Unsplash API].

Python dependencies:

- [requests]
- [vk ] 
- [pyTelegramBotAPI]
